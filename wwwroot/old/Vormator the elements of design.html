Description: Vormator is the ultimate design challenge that requires artists to create a one-page visual by using only eight shapes within a set of given rules.The goal is to show the importance of creativity in terms of dealing with design limitations. Even with a large number of limitations and a few graphical elements to start with, a huge variation of quality graphics is still achievable.
